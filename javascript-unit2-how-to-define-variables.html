ΕΝΟΤΗΤΑ 2 JAVASCRIPT - How to Define Variables

Στο σημερινό δωρεάν μάθημα JavaScript θα μάθουμε πως να ορίζουμε και να ονομάζουμε σωστά τις μεταβλητές όπως και πως αναθέτουμε τιμές σε αυτές.

Όπως όλοι γνωρίζουμε, οι μεταβλητές (variables) έχουν την ιδιότητα να αποθηκεύουν προσωρινά κάποια δεδομένα (αριθμούς, ονόματα, κτλ) και για αυτόν ακριβώς το λόγο, είναι πολύ πιο εύκολο να καλούμε μέσα στο πρόγραμμα μας πολλαπλές φορές μια μεταβλητή παρά την ίδια την τιμή της μεταβλητής που μπορεί να είναι ένας μεγάλος σε μέγεθος αριθμός. 

Φυσικά, κατά την διάρκεια εκτέλεσης του κώδικα, οι τιμές που περιέχουν οι μεταβλητές μπορεί να αλλάξουν αρκετές φορές για αυτό και άλλωστε πήραν το όνομα μεταβλητές (variables). Επίσης μπορούμε να χρησιμοποιήσουμε μεταβλητές και για να αποθηκεύσουμε δεδομένα τα οποία θα μας γίνουν γνωστά μόνο μετά την εκτέλεση του κώδικα όπως π.χ. το αποτέλεσμα μιας αριθμητικής πράξης.

Για να χρησιμοποιήσουμε σωστά μεταβλητές μέσα στον κώδικα μας, πρέπει να ακολουθήσουμε τα δύο πιο κάτω βήματα:

1. Ορίζουμε την μεταβλητή με όνομα

2. Αρχικοποιούμε την μεταβλητή με κάποια τιμή

Στην JavaScript, το όρισμα μιας μεταβλητής απαιτεί την λέξη var στον κώδικα μας (αργότερα θα μάθουμε και για το let και const).

Για παράδειγμα:

var a;

var myvariable;

Για να ονομάσουμε μια μεταβλητή, μπορούμε να χρησιμοποιήσουμε γράμματα, αριθμούς, την κάτω παύλα (underscore) και το σύμβολο του δολαρίου αλλά δεν επιτρέπεται να ξεκινάει το όνομα με αριθμό.

Για να δώσουμε μια αρχική τιμή στην μεταβλητή που έχουμε ήδη ορίσει, υπάρχουν δύο τρόποι:

a. Πρώτα ορίζουμε την μεταβλητή και μετά αναθέτουμε μια τιμή σε αυτήν ή

b.Ορίζουμε και αρχικοποιούμε την μεταβλητή στην ίδια γραμμή κώδικα

Για παράδειγμα:

var myvariable = 5;

Τώρα η μεταβλητή myvariable περιέχει την τιμή 5.

Μπορούμε επίσης να ορίσουμε πολλαπλές μεταβλητές με μια μόνο χρήση της λέξης var αρκεί να ξεχωρίσουμε τις μεταβλητές μεταξύ τους με κόμμα.

Για παράδειγμα:

var v1, v2, v3, v4 = 5, v5;

Οι μεταβλητές είναι case sensitive που σημαίνει ότι η μεταβλητή var a και η μεταβλητή var A θεωρούνται δυο διαφορετικές μεταβλητές για την JavaScript. Μπορούμε όμως να δοκιμάσουμε ένα απλό κώδικα που επιβεβαιώνει αυτή την θεωρία.

Γράψτε τον απλό κώδικα που ακολουθεί μέσα στο αρχείο  MyScript.js και μετά τρέξτε το παράδειγμα.

MyScript.js

 
var x;
x = 5;
document.writeln('the value of x is: ' + x);
document.writeln('<br>');

var check_case = 'lower';
var CHECK_CASE = 'UPPER';

document.writeln('Calling the check_case variable');
document.writeln(check_case);
document.writeln('<br>');
document.writeln('Calling the CHECK_CASE variable');
document.writeln(CHECK_CASE);

Το αποτέλεσμα πρέπει να είναι το εξής:

Σε αυτό το κώδικα, αρχικά ορίζουμε μια μεταβλητή με το όνομα x. Αμέσως μετά της αναθέτουμε την τιμή 5 και ζητάμε να δούμε την τιμή με την οποία ισούται στον browser. Μετά ορίζουμε δύο μεταβλητές που αν και έχουν το ίδιο όνομα η πρώτη είναι γραμμένη με πεζά και η άλλη με κεφαλαία γράμματα. Για την JavaScript αυτές είναι δύο διαφορετικές μεταβλητές και έτσι ακριβώς τις χειρίζεται και η JavaScript.
