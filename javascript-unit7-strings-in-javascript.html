ΕΝΟΤΗΤΑ 7 - Strings In JavaScript

Εισηγητής

ΜΙΧΑΛΗΣ ΚΑΣΑΠΟΓΛΟΥ

Κατηγορία

PROGRAMMING

Στο σημερινό δωρεάν μάθημα JavaScript θα δούμε πως δημιουργούμε Strings, με ποιο τρόπο μπορούν να συνδυαστούν και μερικές χρήσιμες μεθόδους που μας προσφέρει η JavaScript για να τροποποιούμε το αποτέλεσμα τους.

Μια μεταβλητή γίνεται είδος string όταν αναθέτουμε σε αυτήν μια σειρά από χαρακτήρες. Η ανάθεση μπορεί να γίνει χρησιμοποιώντας είτε single quotes ή double quotes.

Δεν υπάρχει καμία διαφορά ανάμεσα στην χρήση των single quotes και double quotes. Ο μόνος λόγος που ίσως επιλέξετε τα double quotes είναι όταν το κείμενο περιέχει απόστροφο. Τότε μπορούμε να χρησιμοποιήσουμε double quotes για να μην θεωρήσει η JavaScript την απόστροφο σαν single quote το οποίο δεν έχει κλείσει σωστά. Μπορούμε επίσης να χρησιμοποιήσουμε και single quote και να δηλώσουμε την απόστροφο σαν escape character. Ο παρακάτω πίνακας σας παρουσιάζει όλους τους escape characters που έχετε διαθέσιμους στην JavaScript.

Όταν χρησιμοποιούμε το σύμβολο της πρόσθεσης ( + ) με τα strings, επειδή δεν είναι δυνατή αριθμητική πράξη με χαρακτήρες, η JavaScript κάνει concatenation δηλαδή ενώνει τα strings μαζί.

Τώρα ίσως αναρωτιέστε τι θα γίνει αν προσθέσω έναν αριθμό σε ένα string? Βασικά δεν θα πάρετε κάποιο λάθος αλλά ένα αποτέλεσμα! Η JavaScript όταν καταλάβει ότι μια από τις δύο τιμές είναι string τότε θα μετατρέψει και τον αριθμό σε string και απλά θα τα ενώσει σαν απλές λέξεις.

Την ίδια λογική μπορείτε να την εφαρμόσετε και σε δύο αριθμούς που απλά θέλετε να ενώσετε χωρίς να τους προσθέσετε. Βάζοντας τους αριθμούς μέσα σε quotes τότε θα γίνει concatenation ανάμεσα τους αντί για πρόσθεση.

Ένας ακόμα συντακτικός τρόπος γραφής των strings που μπορεί να συναντήσετε είναι τα template literals. Τα template literals μας επιτρέπουν να ενσωματώσουμε expressions μέσα στα strings. Αυτή η τεχνική υπάρχει και σε άλλες γλώσσες προγραμματισμού όπως η Python.

Για να μετατρέψουμε ένα string σε ένα template literal πρέπει να αντικαταστήσουμε τα quotes με backtick χαρακτήρες ( ` ` ).

Πριν κλείσουμε το σημερινό δωρεάν μάθημα JavaScript, ας αναφερθούμε και σε μερικές από τις μεθόδους που μας προσφέρει η JavaScript για να τροποποιούμε ή να μορφοποιούμε καλύτερα τα αποτελέσματα των Strings.

Η βασική αρχή εδώ που πρέπει να καταλάβετε είναι ότι τα περισσότερα πράγματα στην JavaScript είναι αντικείμενα. Από την στιγμή που θα δηλώσετε μια μεταβλητή string, αυτόματα αυτή μετατρέπεται σε αντικείμενο και μας προσφέρει μια ομάδα από μεθόδους που μπορούμε να χρησιμοποιήσουμε. Δεν είναι ανάγκη να γνωρίζουμε ακόμα για τα αντικείμενα για να μπορούμε να καταλάβουμε την λειτουργία αυτών των μεθόδων.

Μια από αυτές τις μεθόδους είναι η length( ) που μας επιστρέφει τον συνολικό αριθμό των χαρακτήρων που περιέχονται σε ένα string. Αυτή η μέθοδο μπορεί να χρησιμοποιηθεί σε περιπτώσεις όπου το password του χρήστη έχει πολλούς λιγότερους χαρακτήρες από αυτούς που απαιτούνται.

Επίσης υπάρχουν και η toLowerCase( ) όπως και η toUpperCase( ) οι οποίες μετατρέπουν όλους τους χαρακτήρες του string σε πεζούς ή σε κεφαλαία αντίστοιχα.

Ας δούμε μερικές ακόμα μεθόδους πριν κλείσουμε το δωρεάν μάθημα JavaScript αν και θα επισκεφτούμε αυτές τις μεθόδους αρκετές φορές σε μελλοντικά μαθήματα μας.

Υπάρχει και η μέθοδος indexOf( ) η οποία μας επιστρέφει την θέση μέσα στο string από την οποία ξεκινάει μια συγκεκριμένη συλλαβή η λέξη. Στο παρακάτω παράδειγμα η λέξη author ξεκινάει από την θέση 15 γιατί η αρίθμηση στον προγραμματισμό ξεκινάει από το 0 και όχι το 1.

Τώρα αν θέλουμε να διαβάσουμε την συγκεκριμένη λέξη που ξεκινάει από την θέση 15, μπορούμε χρησιμοποιώντας την μέθοδο slice ( )

Αυτές είναι μόνο μερικές από τις μεθόδους που υπάρχουν διαθέσιμες σε κάθε string. Γράφοντας το όνομα της string μεταβλητής και πατώντας τελεία θα δείτε ολόκληρη την λίστα με μια αναλυτική περιγραφή για κάθε μέθοδο.
