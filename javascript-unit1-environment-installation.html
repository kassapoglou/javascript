PROGRAMMING

Στο σημερινό δωρεάν μάθημα JavaScript θα εγκαταστήσουμε όλα τα απαραίτητα εργαλεία που χρειάζονται για την ανάπτυξη web εφαρμογών χρησιμοποιώντας JavaScript. Επίσης θα δοκιμάσουμε το περιβάλλον μας δημιουργώντας μια απλή html σελίδα η οποία εκτελεί ένα εξωτερικό αρχείο JavaScript και εμφανίζει το αποτέλεσμα στον browser.

Από τις πρώτες κιόλας ημέρες ύπαρξης του διαδικτύου, έγινε αμέσως ορατή η ανάγκη να έχουν οι ιστοσελίδες, και γενικότερα τα  sites, μια πιο δυναμική και διαδραστική συμπεριφορά και να δραπετεύσουν όσο το δυνατόν γρηγορότερα από το στατικό περιβάλλον του HTML. 

Αν και με την παρουσία του CSS μπορέσαμε να δώσουμε ένα πιο όμορφο παρουσιαστικό στα κείμενα μας, με την προσθήκη της JavaScript μπορέσαμε να προσθέσουμε περισσότερες δυνατότητες όπως email, ημερολόγια, τραπεζικές συναλλαγές, online αγορές, παιγνίδια και επεξεργασία κειμένου.

Αν κοιτάξουμε στο παρελθόν, τον τρόπο με τον οποίο καταχωρίσαμε και κατηγοριοποιήσαμε τις τεχνολογίες που κατά καιρούς εμφανίστηκαν, θα δούμε ότι τεχνολογίες όπως η HTML (και όλα τα παρακλάδια της) δημιουργήθηκαν με σκοπό να μας προσφέρουν την ικανότητα να παρουσιάσουμε ένα κείμενο στο διαδίκτυο, τεχνολογίες παρόμοιες με εκείνη της CSS δημιουργήθηκαν για να δώσουμε μια πιο ελκυστική παρουσίαση στο κείμενο μας, και τέλος η JavaScript για να δώσει λειτουργικότητα και να προσφέρει εφαρμογές στον χρήστη του Web.

Τα προγράμματα JavaScript εκτελούνται τοπικά στον υπολογιστή μας. Το περιβάλλον του Web browser είναι η πιο κοινή πλατφόρμα εκτέλεσης JavaScript εφαρμογών αλλά δεν είναι το μόνο. Σήμερα η JavaScript μπορεί να εκτελεστεί σε πολλαπλές πλατφόρμες όπως εκείνων των mobile phones, και των servers.

Η JavaScript είναι το αποτέλεσμα της European Computer Manufactures Association (ECMA) επιτροπής που δημιούργησε το στάνταρ ECMA-262 το οποίο περιγράφει τους βασικούς κανόνες της JavaScript γλώσσας χωρίς καμία αναφορά σε κάποιον συγκεκριμένο  web browser. Αυτό σημαίνει ότι όποια πλατφόρμα (κυρίως  web browsers) θέλει να είναι συμβατή με την τελευταία έκδοση της JavaScript  πρέπει να ακολουθήσει το ECMA στάνταρ.

Το ECMAScript 11 είναι το πιο πρόσφατο στάνταρ και σιγά σιγά όλοι οι γνωστοί web browsers προσπαθούν να το ενσωματώσουν στην πλατφόρμα τους. Η τελευταία έκδοση της JavaScript έρχεται σε εμάς με μια σωρεία από καινούργια χαρακτηριστικά όπως  nullish coalescing operator και globalThis αντικείμενο. Μην ανησυχείτε για αυτά τώρα, θα τα αναλύσουμε όλα αργότερα. Τέλος, μην σας μπερδεύουν τα ονόματα ECMAScript 11, ES11, ή ECMAScript 2020 όλα αναφέρονται στο ίδιο πράγμα.

Εάν δεν είσαστε σίγουροι αν ο browser σας υποστηρίζει ES10 ή ES11  ή κάποια άλλη έκδοση, τότε μπορείτε να επισκεφτείτε την πιο κάτω ιστοσελίδα http://kangax.github.io/compat-table/es2016plus/.

Ας δούμε όμως τώρα πως δημιουργούμε ένα περιβάλλον ανάπτυξης JavaScript κώδικα έτσι ώστε να είμαστε έτοιμοι να τρέξουμε τα παραδείγματα που θα αναλύσουμε ξεκινώντας από την επόμενη ενότητα.

Για αρχή χρειαζόμαστε ένα περιβάλλον ανάπτυξης Web εφαρμογών. Αν και έχουμε πολλές επιλογές, θα προτιμήσουμε την διασημότερη που είναι το Visual Studio Code (https://code.visualstudio.com/) .

Κατεβάστε την τελευταία έκδοση και προχωρήστε στην εγκατάσταση του. Δεν χρειάζεται να το ξεκινήσετε ακόμα.

Τώρα πρέπει να δημιουργήσουμε ένα φάκελλο στον υπολογιστή μας μέσα στον οποίο θα αποθηκεύσουμε τα αρχεία με τον JavaScript κώδικα που πρόκειται να γράψουμε. Μπορείτε να ονομάσετε τον φάκελλο όπως επιθυμείτε. Στο δικό μου παράδειγμα τον ονομάζω JavaScriptLessons.

Τώρα ξεκινάμε το Visual Studio Code. Πατήστε το εικονίδιο Explorer. Από εδώ μπορούμε να επιλέξουμε και να ανοίξουμε τον φάκελλο JavaScriptLessons που έχουμε ήδη δημιουργήσει. Μπορείτε να επιλέξετε τον φάκελο είτε πατώντας επάνω στο κουμπί Open Folder ή μπορείτε να πάτε στο κεντρικό μενού και να επιλέξετε File → Open Folder.

Για να μπορέσουμε να εκτελέσουμε και να δούμε τον κώδικα μας να τρέχει στον Web Browser, χρειάζεται να εγκαταστήσουμε και έναν Web Server. Μην ανησυχείτε όμως για αυτή την διαδικασία γιατί είναι πολύ απλή. Αυτό που έχουμε να κάνουμε είναι να πατήσουμε επάνω στην επιλογή Extensions και να ψάξουμε για το Live Server. Όταν το βρείτε και εμφανιστεί στην λίστα, απλά πατήστε το κουμπί install για να εγκατασταθεί αυτόματα.

Τώρα μπορείτε να πατήσετε πάλι το εικονίδιο Explorer για να γυρίσετε πίσω στον φάκελλο που αντιπροσωπεύει το project μας.

Στο project που δημιουργήσαμε, με βάση τον φάκελο που επιλέξαμε, θέλουμε τώρα να δημιουργήσουμε ένα απλό αρχείο html. Πατάμε επάνω στο κουμπί new file και δημιουργούμε ένα αρχείο  index.html. Όταν δημιουργείτε αρχεία βεβαιωθείτε ότι γράφετε όχι μόνο το όνομα αλλά και την κατάληξη του αρχείου π.χ. html. Το Visual Studio Code ενεργοποιεί τα modules που περιέχει για να υποστηρίξει το project μας βασιζόμενο στις καταλήξεις των αρχείων.

Ένα απλό html αρχείο χρειάζεται κάποιο βασικό κώδικα για να μπορεί να τρέξει σωστά στον browser. Δεν χρειάζεται όμως να δημιουργείτε αυτόν τον βασικό κώδικα από την αρχή κάθε φορά. Το Visual Studio Code θα σας βοηθήσει να το δημιουργήσετε αυτόματα.

Γράψουμε μέσα στο index.html αρχείο την λέξη doc. Η επιλογή που μας εμφανίζεται, μας προτείνει την δημιουργία ενός βασικού κορμού μιας html σελίδας. Πατήστε επάνω στην επιλογή doc που εμφανίζεται και θα δείτε αμέσως όλο τον κώδικα html να γράφεται αυτόματα.

Πριν προσθέσουμε JavaScript κώδικα σε αυτή την απλή html σελίδα, ας εξοικειωθούμε λίγο με την διαδικασία εκτέλεσης ενός html αρχείου μέσα από το Visual Studio Code. Για αρχή ας προσθέσουμε ένα απλό μήνυμα μέσα στο body tag έτσι ώστε να δούμε κάτι στον browser όταν αυτή η σελίδα τρέξει στον Live Server. Προσθέτουμε λοιπόν το εξής μήνυμα:

Αν δεν σας εμφανίζεται, απλά κλείστε το Visual Studio Code και ξεκινήστε το πάλι για να αναγνωρίσει την εγκατάσταση του συγκεκριμένου extension.

Τώρα πατήστε πάλι επάνω στο εικονίδιο Go Live για να σταματήσει ο Live Server. Όταν προσθέσουμε και τoν JavaScript κώδικα τότε θα τον ενεργοποιήσουμε πάλι.

Για προσθέσουμε JavaScript κώδικα στην html σελίδα μας, υπάρχουν δύο τρόποι – ο πρώτος είναι να τον προσθέσουμε μέσα στον html κώδικα (inline) ή σε εξωτερικό αρχείο (external). Ο καλύτερος τρόπος είναι να έχουμε τον JavaScript κώδικα σε εξωτερικό αρχείο γιατί είναι πιο γρήγορη η εκτέλεση του. Θα μιλήσουμε εκτενέστερα και για τις δύο αυτές επιλογές στην επόμενη ενότητα. Για τώρα ας δούμε που γράφουμε τον JavaScript κώδικα και πως μπορούμε να δούμε το αποτέλεσμα στον browser.

Δημιουργούμε λοιπόν ένα δεύτερο αρχείο μέσα στο φάκελλο μας με το όνομα myscript.js. Η κατάληξη .js δηλώνει ότι το συγκεκριμένο αρχείο θα περιέχει JavaScript κώδικα. Γράφουμε λοιπόν την εξής γραμμή κώδικα που θα μας δείξει ένα μήνυμα στην κονσόλα του browser.

Τα περισσότερα μας JavaScript βασικά παραδείγματα θα τα εκτελούμε μέσα από την κονσόλα του Browser. Θα ακολουθήσουμε αυτή τη μέθοδο για λίγο μέχρι να εξοικειωθούμε με τις JavaScript εντολές και μετά θα δημιουργήσουμε web εφαρμογές.

Αυτό λοιπόν είναι το περιβάλλον που θα χρειαστούμε για να τρέξουμε όλα τα παραδείγματα από τις μελλοντικές ενότητες των δωρεάν μαθημάτων JavaScript. Απλά θα αλλάζουμε τον κώδικα μέσα στο αρχείο myscript.js και θα κάνουμε refresh στην ιστοσελίδα για να δούμε το αποτέλεσμα.
