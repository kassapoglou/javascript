ΕΝΟΤΗΤΑ 5 JAVASCRIPT - String Data Type

Στο σημερινό δωρεάν μάθημα JavaScript θα περιγράψουμε τι είναι τα Strings και θα δούμε αναλυτικά κάποια απλά παραδείγματα που δείχνουν την λειτουργία τους και την χρησιμότητα τους.

Ένα string είναι μια σειρά από χαρακτήρες που όλοι μαζί ανήκουν στην ίδια λέξη ή έκφραση. Στην JavaScript, οτιδήποτε (γράμματα, αριθμοί, σύμβολα, κτλ) βάλουμε ανάμεσα σε single quotes (‘ ’) ή σε double quotes(“ ”) θεωρείται να είναι είδος string.

Αυτό σημαίνει ότι ο αριθμός 1 μέσα σε double quotes (“1”) θεωρείται σαν ένα string που περιέχει τον χαρακτήρα 1.

Με βάση τα προηγούμενα παραδείγματα που έχουμε τρέξει, έχετε ήδη καταλάβει ότι μπορούμε να χρησιμοποιήσουμε το typeof τελεστή για να βεβαιωθούμε για το είδος του data που περιέχει μια μεταβλητή.

MyScript.js


var s = "Michail";
document.writeln(typeof s);
document.writeln('</BR');
var x = "1";
document.writeln(typeof s);

output

	 
string 
string

Εάν δεν βάλουμε τίποτα ανάμεσα στα double quotes (“ ”), το αποτέλεσμα είναι επίσης string.

MyScript.js


var s = "";
document.writeln(typeof s);

output


string

Όπως ήδη γνωρίζετε, το σύμβολο της πρόσθεσης (+) εκτελεί την αριθμητική πράξη της πρόσθεσης ανάμεσα σε αριθμούς. Εάν όμως το χρησιμοποιήσετε ανάμεσα σε δύο ή περισσότερα strings τότε αυτό που θα έχουμε σαν αποτέλεσμα δεν θα είναι αριθμητική πρόσθεση αλλά το “κόλλημα” του ενός string δίπλα στο άλλο. Αυτή η διαδικασία ονομάζεται concatenation. Δηλαδή σκεφτείτε ότι έχουμε μια μεταβλητή x1 = “Michail” και μια δεύτερη x2 = “Kassapoglou”. Η χρήση του συμβόλου ( + ) ανάμεσα στα δύο strings (x1 + x2) θα δώσει ένα καινούργιο string που θα έχει τιμή “MichailKassapoglou”. Αν θέλετε να έχετε κενό ανάμεσα στις δύο λέξεις στο τελικό αποτέλεσμα, τότε θα πρέπει να ορίσετε ένα κενό ανάμεσα στο τελευταίο χαρακτήρα του string και του quote π.χ “Michail ”.

MyScript.js



var firstName = "Michail ";
var lastname = "Kassapoglou";
var firstLast = firstLast + lastname;
document.writeln(firstLast);

output


Michail Kassapoglou

Αυτός ο διπλός ρόλος του συν ( + ) τελεστή είναι η ρίζα πολλών προβλημάτων σε κώδικα γραμμένο σε JavaScript. Για αυτό καλό θα ήταν να κάνετε έναν έλεγχο πρώτα αν όλες οι τιμές που προσπαθείτε να προσθέσετε είναι όλες αριθμοί ή όλες string για να αποφύγετε απρόσμενα αποτελέσματα.

Ο λόγος που προτείνουμε τον παραπάνω έλεγχο είναι επειδή όταν η JavaScript δει ένα string που μοιάζει με αριθμό, όπως π.χ “1” (όχι όμως γραμμένο με χαρακτήρες όπως “one”) τότε το string “1” μετατρέπεται αυτόματα στον αριθμό 1 για όλες τις αριθμητικές πράξεις εκτός από την πρόσθεση (λόγω του διπλού ρόλου που έχει). Ένας γρήγορος τρόπος (αν και όχι τόσο προτεινόμενος) για γρήγορη μετατροπή μιας string τιμής σε πραγματικό αριθμό είναι ο πολλαπλασιασμός της τιμής string με τον αριθμό 1. Η αντίστροφη διαδικασία (δηλαδή μετατροπή ενός αριθμού σε string) γίνεται αν προσθέσουμε στον αριθμό ένα άδειο string – δηλαδή τα double quote χωρίς να περιέχουν κάποια τιμή.

MyScript.js


var s = "2"
var k = 3;
var x = s * k;
document.writeln(x);
document.writeln('</BR>')
var i = "5" * 1;
document.writeln(i);
document.writeln('</BR>');
var n = 1;
n = "" + n;
document.writeln(typeof n);

output


6
5
string

Αν πάμε στο https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String όπου υπάρχει ένα υπέροχο documentation της JavaScript, θα δούμε έναν πίνακα που περιέχει strings με ειδικά σύμβολα που μπορούμε να ενσωματώσουμε στον κώδικα μας.

Πριν κλείσουμε αυτή την ενότητα, θα αναφερθούμε και σε ένα καινούργιο χαρακτηριστικό που προστέθηκε με την ES6 και ονομάζεται template literals. Τα template literals επιτρέπουν μεταβλητές και πράξεις να ενσωματωθούν με το τελικό string output που προσπαθούμε να δημιουργήσουμε. Πριν δείξουμε ένα παράδειγμα, πρέπει να αναφέρουμε το γεγονός ότι τα template literals πρέπει να περικλείονται μέσα σε back-ticks(` `) τα οποία βρίσκονται επάνω αριστερά στο πληκτρολόγιο δίπλα από τον αριθμό 1. Τέλος, για να ενσωματώσουμε template literals μέσα στον κώδικα μας, πρέπει να χρησιμοποιήσουμε το σύμβολο του δολαρίου και άγκιστρα (${exression}).

Ας δούμε όμως ένα παράδειγμα, για να καταλάβουμε καλύτερα την χρησιμότητα των template literals.

MyScript.js


var a = 10;
var b = 10;
document.writeln(`Sum is ${a+b} and Multiplication would be ${a*b}`);

output

		
Sum is 20 and Multiplication would be 100

Αν θέλετε να βρείτε περισσότερες πληροφορίες για τα template literals μπορείτε να πάτε στο https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Template_literals όπου θα βρείτε αρκετά παραδείγματα.
