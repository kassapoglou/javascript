ΕΝΟΤΗΤΑ 6 - Numbers And Operators

Εισηγητής

ΜΙΧΑΛΗΣ ΚΑΣΑΠΟΓΛΟΥ

Κατηγορία

PROGRAMMING

Στο σημερινό δωρεάν μάθημα JavaScript θα αναλύσουμε πως χρησιμοποιούμε τους operators για να πραγματοποιήσουμε αριθμητικές πράξεις.

Οι operators δέχονται μια ή περισσότερες τιμές (ή μεταβλητές), εκτελούν μια πράξη και μας επιστρέφουν πίσω ένα αποτέλεσμα. Οι operators χωρίζονται σε αρκετές κατηγορίες. Για αρχή ας ασχοληθούμε με τους arithmetic operators.

Στο παρακάτω απλό παράδειγμα προσθέτουμε δύο αριθμούς με το + operator.

Τις περισσότερες φορές όμως δεν κάνουμε πράξεις με αριθμούς αλλά με τις μεταβλητές στις οποίες έχουμε αναθέσει τους αριθμούς.

Πριν προχωρήσουμε πρέπει εξηγήσουμε ότι και το σύμβολο = στο statement let num1 = 5; είναι operator και ονομάζεται simple assignment operator.

Τώρα, όσον αφορά τα είδη των αριθμών που μπορεί να διαχειριστεί η JavaScript εκεί δεν έχουμε κανένα περιορισμό όμως όλα τα είδη των αριθμών η JavaScript τα διαχειρίζεται σαν numbers.

Το typeof είναι και αυτός ένας operator που επιστρέφει σαν string το είδος της τιμής που έχει μια μεταβλητή. Αυτός είναι ένας χρήσιμος operator σε μια dynamic typed language όπως είναι η JavaScript όπου οι μεταβλητές αλλάζουν data type ανάλογα με τις τιμές που τις αναθέτουμε.

Κάθε ένα από τα primitive type (εκτός από το null και undefined) έχει και τα αντίστοιχα αντικείμενα όπως Number για το number primitive, String για το string primitive κτλ. Τι σημαίνει αυτό? Αυτό σημαίνει ότι αφού ορίσουμε μια μεταβλητή σαν number έχουμε ορισμένες μεθόδους διαθέσιμες για να μορφοποιήσουμε το αποτέλεσμα.

Επίσης μπορούμε να μετατρέψουμε ένα string σε number αρκεί το string να έχει γραφτεί με αριθμούς.

Πριν προχωρήσουμε την θεωρία μας ας δούμε συνολικά πως λειτουργούν τα operators μέσα από ένα απλό παράδειγμα.

Όταν κάνουμε αριθμητικές πράξεις και συνδυάζουμε πολλά arithmetic operators στο ίδιο statement, η JavaScript ακολουθεί μια συγκεκριμένη σειρά με την οποία θα εκτελέσει τις πράξεις. Κάποιες αριθμητικές πράξεις προηγούνται και έχουν μεγαλύτερη προτεραιότητα από κάποιες άλλες. Για παράδειγμα, αν στο ίδιο statement υπάρχει η πράξη της πρόσθεσης και η πράξη του πολλαπλασιασμού, τότε πρώτα θα εκτελεστεί ο πολλαπλασιασμός και μετά η πρόσθεση. Στον παρακάτω πίνακα μπορείτε να βρείτε αναλυτικά τους operators και την σειρά εκτέλεσής τους.

Στο παραπάνω παράδειγμα, πρώτα εκτελέστηκε η διαίρεση και μετά η πρόσθεση των αριθμών. Αν θέλετε να ορίσετε εσείς την προτεραιότητα εκτέλεσης των πράξεων τότε μπορείτε να τις βάλετε σε παρενθέσεις. Ότι βρίσκεται μέσα σε παρενθέσεις εκτελείται πρώτο. Αν οι πράξεις έχουν όλες την ίδια προτεραιότητα τότε ακολουθείται η αλγεβρική σειρά εκτέλεσης – δηλαδή από τα αριστερά προς τα δεξιά.

Πολλές φορές, χρειάζεται να αυξήσουμε έναν αριθμό κατά ένα ή να τον μειώσουμε κατά ένα. Για αυτόν ακριβώς το σκοπό υπάρχουν οι increment ( ++ ) και οι decrement ( -- ) operators τους οποίους εφαρμόζουμε πριν το όνομα της μεταβλητής (prefix) ή μετά το όνομα της μεταβλητής (postfix).

Μέσα στην ομάδα των operators υπάρχουν και οι assignment operators που δουλειά τους είναι να αναθέτουν τιμές στις μεταβλητές. Υπάρχει όμως μια συντομογραφία που πολύ συχνά χρησιμοποιείται και είναι η εξής:

Τέλος υπάρχουν και οι comparison operators οι οποίοι χρησιμοποιούνται για συγκρίσεις αριθμών. Τους comparison operators, όπως αργότερα και τους Boolean operators, θα τους χρησιμοποιήσουμε όταν προσθέσουμε στην θεωρία μας και τα if statements.
