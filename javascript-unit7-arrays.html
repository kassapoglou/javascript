ΕΝΟΤΗΤΑ 7 JAVASCRIPT - Arrays

Στο σημερινό δωρεάν μάθημα JavaScript θα δούμε προχωρημένες μεθόδους διαχείρισης και αποθήκευσης δεδομένων που ονομάζονται arrays.

Τι είναι ένας array? Με απλά λόγια, είναι μια λίστα από τιμές. Αντί να χρησιμοποιούμε κάθε φορά μια μεταβλητή για κάθε τιμή που πρέπει να αποθηκεύσουμε, μπορούμε να χρησιμοποιήσουμε μια και μόνο μεταβλητή για να αποθηκεύσουμε πολλαπλές τιμές. Οι τιμές αυτές αποτελούν τα elements του array όπως ονομάζονται.

Για να ορίσουμε έναν array (που αρχικά δεν περιέχει καμία τιμή), αρκεί να ορίσουμε μια μεταβλητή (όπως ήδη γνωρίζουμε) που να ισούται με κενά brackets σύμβολα:

var a = [ ];

Τώρα αν θέλουμε να ορίσουμε έναν array που περιέχει τρεις τιμές, δεν έχουμε παρά να γράψουμε τις τιμές μέσα στα brackets. Με κόμμα χωρίζουμε την μια τιμή από την επόμενη.

var a = [1, 2, 3];

Τώρα αν θέλουμε να δούμε τις τιμές που περιέχει ο array, μπορούμε να καλέσουμε το όνομα της μεταβλητής που τον ορίζει. Ας δούμε ένα παράδειγμα.

MyScript.js


var a = [1, 3, 5];
document.writeln(a);

output

	 
1,3,5

Τώρα η ερώτηση είναι πως μπορούμε να αποκτήσουμε πρόσβαση σε κάθε μια από τις τιμές ξεχωριστά που είναι αποθηκευμένες στον array? Κάθε array λοιπόν χαρακτηρίζεται από έναν index αριθμό που ξεκινάει την αρίθμηση του από το μηδέν και μεγαλώνει όσο μεγαλώνει και ο array. Κατά συνέπεια, και τα στοιχεία που περιέχονται μέσα στον array είναι αριθμημένα από έναν index αριθμό. Στον array που μόλις δημιουργήσαμε στο προηγούμενο παράδειγμα, το πρώτο στοιχείο έχει index 0, το δεύτερο στοιχείο έχει index 1 και το τρίτο στοιχείο έχει index 2.

Ακόμα όμως δεν έχουμε απαντήσει την ερώτηση πως αποκτούμε πρόσβαση σε κάθε ένα από τα στοιχεία του array. Τώρα που γνωρίζουμε τι είναι το index, για να αποκτήσουμε πρόσβαση σε οποιοδήποτε στοιχείο του array δεν έχουμε παρά να καλέσουμε τον index αριθμό μέσα σε brackets. Για παράδειγμα, αν θέλαμε να βρούμε πια είναι η τιμή που περιέχεται μέσα στο index 0 δεν έχουμε παρά να καλέσουμε το όνομα του array με τον αριθμό του index μέσα σε brackets.

a[0];

MyScript.js


var a = [1, 3, 5];

document.writeln("The value inside index 0 is: " + a[0]);
document.writeln('</BR>');
document.writeln("The value inside index 1 is: " + a[1]);
document.writeln('</BR>');
document.writeln("The value inside index 2 is: " + a[2]);

output

	 
The value inside index 0 is: 1
The value inside index 1 is: 3
The value inside index 2 is: 5

Με την χρήση του index, μπορούμε επίσης να ανανεώσουμε τις ήδη υπάρχουσες τιμές ή να προσθέσουμε καινούργιες τιμές στον array. Για παράδειγμα, ας αλλάξουμε την τιμή του index [2] από 5 σε string data type με την τιμή “ten”.

MyScript.js


var a = [1, 3, 5];
a[2] = "ten";

document.writeln("The value inside index 0 is: " + a[0]);
document.writeln('</BR>');
document.writeln("The value inside index 1 is: " + a[1]);
document.writeln('</BR>');
document.writeln("The value inside index 2 is: " + a[2]);

output

	 
The value inside index 0 is: 1
The value inside index 1 is: 3
The value inside index 2 is: ten

Μπορούμε επίσης να προσθέσουμε πιο περισσότερα elements σε έναν index που δεν υπήρχε προηγουμένως. Για παράδειγμα, θέλουμε να προσθέσουμε μια καινούργια τιμή στο index a[3].

MyScript.js


var a = [1, 3, 5];
a[2] = "ten";
a[3] = 200;

document.writeln("The value inside index 0 is: " + a[0]);
document.writeln('</BR>');
document.writeln("The value inside index 1 is: " + a[1]);
document.writeln('</BR>');
document.writeln("The value inside index 2 is: " + a[2]);
document.writeln('</BR>');
document.writeln("The value inside index 3 is: " + a[3]);

output

	 
The value inside index 0 is: 1
The value inside index 1 is: 3
The value inside index 2 is: ten
The value inside index 3 is: 200

Μπορείτε να μαντέψετε πως να διαγράφετε στοιχεία από τον array? Μα φυσικά πάλι με την χρήση του index αλλά με συνδυασμό με την εντολή delete. Προσοχή λιγάκι με την θεωρία εδώ – μετά από την διαγραφή του στοιχείου, το μήκος του array δεν αλλάζει, δηλαδή δεν μειώνεται. Στην ουσία έχουμε έναν array που θα έχει “undefined” σαν τιμή για ένα ή και περισσότερα στοιχεία που έχουν διαγραφεί.

MyScript.js


var a = [1, 3, 5];
a[2] = "ten";
a[3] = 200;
delete a[2];

document.writeln("The value inside index 0 is: " + a[0]);
document.writeln('</BR>');
document.writeln("The value inside index 1 is: " + a[1]);
document.writeln('</BR>');
document.writeln("The value inside index 2 is: " + a[2]);
document.writeln('</BR>');
document.writeln("The value inside index 3 is: " + a[3]);

output

	 
The value inside index 0 is: 1
The value inside index 1 is: 3
The value inside index 2 is: undefined
The value inside index 3 is: 200

Ένας array μπορεί να περιέχει όλων των ειδών τα στοιχεία συμπεριλαμβανόμενου και άλλου array.

MyScript.js


var a = [1, 3, 5, 7, 9];
a[5] = [10, 12, 12];

document.writeln("The array a has the following elements: " + a);

output

	 
The array a has the following elements: 1,3,5,7,9,10,12,12

Αν και νομίζω ότι έχετε ήδη καταλάβει πως ορίζεται ένας τέτοιου είδους array, ας δούμε ένα ακόμα παράδειγμα όπου και τα δύο στοιχεία ενός array είναι array.

MyScript.js


var a = [[1,2,3], [4,5,6]];

document.writeln("The array a has the following elements: " + a);

output

	 
The array a has the following elements: 1,2,3,4,5,6

Στο πιο πάνω παράδειγμα, το πρώτο element του array a είναι ο array με στοιχεία [1,2,3] ενώ το δεύτερο element είναι ο array με στοιχεία [4,5,6]. Για να αποκτήσουμε πρόσβαση σε αυτό του είδους τα array (που ονομάζονται nested arrays) χρησιμοποιούμε διπλά σετ από brackets. Για παράδειγμα, για να πάρουμε την πρώτη τιμή από τον δεύτερο array (που είναι και το δεύτερο element του array a) γράφουμε a[1][0]. Όπου [1] υποδηλώνει ότι θέλουμε να πάμε στο δεύτερο στοιχείο του array a, και επειδή αυτό το στοιχείο είναι array θέλουμε το πρώτο του στοιχείο [0].

MyScript.js


var a = [[1,2,3], [4,5,6]];

document.writeln("The array a has the following elements: " + a);
document.writeln('</BR>');
document.writeln("The element inside the index a[1][0] is: " + a[1][0]);

output

	 
The array a has the following elements: 1,2,3,4,5,6
The element inside the index a[1][0] is: 4

Υπάρχουν πολλά περισσότερα να μάθουμε για τους arrays αλλά θα περιμένουμε πρώτα μέχρι να μιλήσουμε για αντικείμενα και μετά θα επισκεφτούμε τους array πάλι για να μιλήσουμε για τις επιπλέον ιδιότητες που έχουν.
