ΕΝΟΤΗΤΑ 8 - Arrays

Εισηγητής

ΜΙΧΑΛΗΣ ΚΑΣΑΠΟΓΛΟΥ

Κατηγορία

PROGRAMMING

Στο σημερινό δωρεάν μάθημα JavaScript θα εξηγήσουμε τι είναι ένας array, πως δημιουργούμε arrays, πως προσθέτουμε και αφαιρούμε στοιχεία από αυτούς και κάποιες έξτρα ιδιότητες τους που μπορούμε να χρησιμοποιήσουμε καλώντας τις μεθόδους που περιέχουν. Τι είναι όμως ένας array?

Με απλά λόγια είναι μια λίστα από τιμές. Αντί να χρησιμοποιούμε κάθε φορά μια μεταβλητή για κάθε τιμή που πρέπει να αποθηκεύουμε, μπορούμε να χρησιμοποιήσουμε μια και μόνο μεταβλητή για να αποθηκεύσουμε πολλαπλές τιμές. Οι τιμές αυτές αποτελούν τα elements του array όπως ονομάζονται. Σκεφτείτε λοιπόν ότι έχετε 10 διαφορετικές τιμές να αποθηκεύσετε, ίσως να μην σας φανούν πολλές οπότε μπορεί να αποφασίσετε να δημιουργήσετε 10 διαφορετικές μεταβλητές για να τις αποθηκεύσετε. Τι θα κάνετε όμως αν έχετε 100 διαφορετικές τιμές ή 1000 διαφορετικές τιμές? Εδώ είναι ακριβώς που χρειαζόμαστε τους array.

Για να ορίσουμε αρχικά έναν array, αρκεί να ορίσουμε μια μεταβλητή που με ισούται με κενά brackets ( [ ] ). Αν ο array δεν περιέχει καμία τιμή το αποτέλεσμα θα δείχνει length 0.

Τώρα, αν θέλουμε να ορίσουμε έναν array που περιέχει τιμές, δεν έχουμε παρά να γράψουμε αυτές τις τιμές μέσα στα brackets. Χωρίζουμε την μια τιμή από την άλλη με κόμμα. Για να δούμε τις τιμές που περιέχει ο array μπορούμε να καλέσουμε το όνομα της μεταβλητής που τον ορίζει.

Στο παραπάνω παράδειγμα, όλα τα δεδομένα του array είναι string. Όμως σε έναν array μπορούμε να αποθηκεύσουμε πολλά διαφορετικά data types όπως numbers, objects κτλ όπως και συνδυασμούς τους.

Η ερώτηση που σίγουρα θα έχετε τώρα στο μυαλό σας είναι η εξής: «Πως μπορώ να αποκτήσω πρόσβαση σε κάθε μια από τις τιμές ξεχωριστά?». Κάθε array λοιπόν χαρακτηρίζεται από έναν index αριθμό που ξεκινάει την αρίθμησή του από το μηδέν και μεγαλώνει κατά ένα όσο μεγαλώνει και ο array. Στο shopping array που δημιουργήσαμε στο προηγούμενο παράδειγμα, το πρώτο στοιχείο έχει index 0, το δεύτερο στοιχείο έχει index 1, το τρίτο στοιχείο έχει index 2 κτλ.

Ακόμα όμως δεν έχουμε απαντήσει στην ερώτηση πως αποκτούμε πρόσβαση σε κάθε ένα από τα στοιχεία του array. Τώρα που γνωρίζουμε τι είναι το index, για να αποκτήσουμε πρόσβαση σε οποιοδήποτε στοιχείο ενός array δεν έχουμε παρά να καλέσουμε τον index αριθμό μέσα σε brackets. Για παράδειγμα, αν θέλουμε να βρούμε ποια είναι η τιμή που περιέχεται μέσα στον index 0 δεν έχουμε παρά να καλέσουμε το όνομα του array με τον αριθμό του index μέσα σε brackets.

Με την χρήση του index μπορούμε να αντικαταστήσουμε τιμές που ήδη υπάρχουν μέσα στον array ή και ακόμα να προσθέσουμε καινούργιες τιμές.

Όπως υπάρχει στα Strings, έτσι και στα arrays βρίσκουμε το property length που μας φανερώνει πόσα στοιχεία υπάρχουν μέσα στον array.

Μπορείτε να μαντέψετε πως διαγράφουμε στοιχεία από τον array? Μα φυσικά και πάλι με την βοήθεια του index σε συνδυασμό με την εντολή delete. Προσοχή λιγάκι με την θεωρία εδώ – μετά από την διαγραφή του στοιχείου, το μήκος του array δεν αλλάζει, δηλαδή δεν μειώνεται. Στην ουσία έχουμε έναν array που θα έχει “undefined” σαν τιμή για ένα ή περισσότερα στοιχεία που έχουν διαγραφεί.

Ένας array μπορεί να περιέχει όλων των ειδών τα στοιχεία συμπεριλαμβανομένου και άλλου array

Αν και νομίζω ότι έχετε ήδη καταλάβει πως ορίζεται ένας τέτοιου είδους array, ας δούμε ένα ακόμα παράδειγμα όπου και τα δύο στοιχεία ενός array είναι array.

Στο πιο πάνω παράδειγμα, το πρώτο element του array a είναι ο array με στοιχεία [1,2,3] ενώ το δεύτερο element είναι ο array με στοιχεία [4,5,6]. Για να αποκτήσουμε πρόσβαση σε αυτό του είδους τα array (που ονομάζονται nested arrays) χρησιμοποιούμε διπλά σετ από brackets. Για παράδειγμα, για να πάρουμε την πρώτη τιμή από τον δεύτερο array (που είναι και το δεύτερο element του array a) γράφουμε a[1][0]. Όπου [1] υποδηλώνει ότι θέλουμε να πάμε στο δεύτερο στοιχείο του array a, και επειδή αυτό το στοιχείο είναι array θέλουμε το πρώτο του στοιχείο [0].

Ένας array περιέχει έναν μεγάλο αριθμό μεθόδων που μας βοηθάνε με την διαχείριση των δεδομένων του array. Μια όμως από αυτές τις μεθόδους ξεχωρίζει γιατί αν και ανήκει στα strings το αποτέλεσμα της είναι η δημιουργία ενός array από τα στοιχεία ενός string. Η μέθοδο ονομάζεται split( ) και μέσα στην παρένθεση αυτό που δέχεται είναι απλά ο χαρακτήρας ο οποίος θα χρησιμοποιηθεί για να σπάσει το string σε μεμονωμένα στοιχεία μέσα σε έναν array.

Μπορούμε επίσης να κάνουμε ακριβώς την αντίθετη διαδικασία, δηλαδή από έναν array να δημιουργήσουμε ένα string. Αυτό είναι εφικτό με την μέθοδο join( ).

Ένας ακόμα τρόπος για να μετατρέψουμε έναν array σε string είναι να χρησιμοποιήσουμε την toString( ) μέθοδο. Αν και είναι λίγο πιο εύκολη στην μετατροπή ενός array σε string, γιατί δεν δέχεται παραμέτρους, από την άλλη όμως έχει περιορισμούς στις δυνατότητες της. Για παράδειγμα στην μετατροπή πάντα χρησιμοποιεί το κόμμα ( , ) και δεν μπορούμε να αλλάξουμε αυτή την συμπεριφορά.

Πριν κλείσουμε το σημερινό δωρεάν μάθημα JavaScript, θα αναφερθούμε και σε τέσσερις ακόμα μεθόδους: την push( ) που προσθέτει δεδομένα στο τέλος του array, την pop( ) που αφαιρεί στοιχεία από το τέλος του array, την unshift( ) που προσθέτει στοιχεία στην αρχή του array και την shift( ) που αφαιρεί στοιχεία από την αρχή του array.
